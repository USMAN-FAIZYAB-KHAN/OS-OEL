1. Compiled File
The name of the compiled file is "dynamic_mem_allocation".

2. Command To Compile:
gcc -o dynamic_mem_allocation main.c display.c memory.c process.c

3. Input:
The program takes input from the command line.
(The units used when providing process or memory size is case sensitive MB != mB)

4. Output:
Output appears on the console but an "output.txt" is also generated by the program because the console can show limited output.

5. Files:
-> "display.c" contains functions for displaying memory and tables on the console.
-> "process.c" contains structures (e.g. queues) and functions related to process.
-> "memory.c" contains structures and functions related like memory allocation, deallocation, compaction.
-> "main.c" contains the main program (simulation).
